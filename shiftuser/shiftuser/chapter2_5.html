Getting Started with SHIFT

This chapter introduces a new user to the SHIFT facilities. SHIFT is batch processing facility for physics analysis and user submit jobs to SHIFT using NQS. Interactive login on the SHIFT machines is also allowed and is used for program development and small test jobs. Any production work should be run in batch mode as long interactive jobs interfere with job scheduling on the systems.

Getting an Account

SHIFT accounts are allocated through the normal group administrators and you you first check that your group has CPU resources on SHIFT. The group administrator uses USERREG exec on CERNVM and enter the SHIFT panel. Please note that only one group code is allowed per UNIX username.

Passwords

Upon registration for SHIFT, users are assigned an initial password. To change your password use the command passwd (a link to /usr/bin/yppasswd). This will prompt for your current NIS password and then ask for two copies of your new choice. NIS stands for Network Information Service and is the registry used on SHIFT. (NIS is also known as Yellow Pages).

There is no password ageing on SHIFT but users are recommended to use non-trivial passwords. At least 6 characters should be used and at least one non alphanumeric character should be included in the password. Avoid any dictionary word and any simple name associated with you or your login identifier.

Login and Default Shell Environment

To login to SHIFT, users should telnet or rlogin to an appropriate system: SHIFT1 in the case of OPAL, and SHIFT6 in the case of ALEPH. User accounts are setup with default .cshrc, .profile and .login files. All groups are also setup with a pubgg directory with default group_cshrc and group_profile files.

The default shell is generally the csh but the in case of ALEPH, tcsh is used. Users who have their own workstations may want to start up an xterm on their own display from csf. To do this, type 




  xhost shift1

on your own workstation. Then login into your SHIFT node and type 




  /usr/bin/X11/xterm  -display yourhost:0 &

Getting Online Help and Documentation

  • All man pages are installed on SHIFT. To get a full description of a command, type 

    


  man command.

  • To get a list of commands related to a particular keyword, type 

    


  apropos keyword.

  • To discover where commands are located, type 

    


  whereis command

Printing files on SHIFT

The UNIX lpr/lpd line printer spooling system is installed on the SHIFT. Printers known to the system are defined in the file /etc/printcap

To print a file, type the following command 




  lpr -P
