Examples for HistOOgram and HepODBMS

Examples for the histOOgram and HepODBMS packages are available in the directories:

Unix: /afs/cern.ch/sw/lhcxx/share/HISTOOGRAMS/pro/examples
      /afs/cern.ch/sw/lhcxx/share/HepODBMS/pro/HepODBMS/examples
Nice: Z:\P32\lhcxx\SHARE\HISTOOGRAMS\pro\examples\
      Z:\P32\lhcxx\SHARE\HEPODBMS\pro\HepODBMS\examples\

Each example has its own subdirectory, containing the C++ source code in a file with extension .cpp, as well as a GNUmakefile which allows you to generate an executable with gmake.
