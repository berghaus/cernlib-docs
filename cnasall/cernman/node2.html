The user commands

In this first part we describe sectioning and simple text generating commands, with or without argument, which you can use with the cernman  style.

For the layout cernman  uses LaTeX's standard report  style. This has the advantage that the present style file is more robust with respect to future internal changes to the LaTeX styles.
