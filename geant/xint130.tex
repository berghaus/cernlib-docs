\Makehead{The Geometry Commands}
\Ssubr{SVOLU  name shape numed par npar}
{\tt CALL GSVOLU (name,shape,numed,par,npar,ivolu)}
 where par is a {\tt ZCEDEX} vector
\Ssubr{SPOS  name number mother x0 y0 z0 irot only }
{\tt  CALL GSPOS (name,number,mother,x0,y0,z0,irot,only)}
\Ssubr{SDVN  name mother ndiv iaxis }
{\tt  CALL GSDVN (name,mother,ndiv,iaxis)}
\Ssubr{SROTM  irot theta1 phi1 theta2 phi2 theta3 phi3 }
{\tt CALL GSROTM (irot,theta1,phi1,theta2,phi2,theta3,phi3)}
\Ssubr{STMED ntmed name nmat isvol ifield fieldm tmaxfd tmaxms epsil}
 
\Sbox{CALL GSTMED (ntmed,name,nmat,isvol,ifield,fieldm,tmaxfd,\\
dmaxms,deemax,epsil,stmin,0,0)}
 
Defaults are :
\begin{center}
\tt ntmed=1  nmat=1  isvol=0  ifield=0  fieldm=0.\\
tmaxfd=0.01  dmaxms=0.01  deemax=0.01  epsil=0.01
\end{center}
\Ssubr{EDITV  [isel]}
{\tt CALL GEDITV (isel)}
 
If {\tt ISEL} is not given, the menu is shown.
When the routine asks for input parameters that have not to
be changed, the carriage return must be typed.
{\tt ISEL} is used to select the editing operation to be performed:
\begin{DL}{MMMM}
\item[ISEL=1] to modify shape parameters {\tt PAR} given by \Rind{GSVOLU}
\item[ISEL=2] to modify {\tt NAME} given by \Rind{GSVOLU}
\item[ISEL=3] to delete {\tt NAME} given by \Rind{GSVOLU}
\item[ISEL=4] to unlink {\tt NAME, NR} given by \Cind{GSPOS/GSDVN/GSDV}
\item[ISEL=5] to modify {\tt X0, Y0, Z0} of {\tt NAME, NR} given by \Rind{GSPOS}
\item[ISEL=6] to modify {\tt IROT} of {\tt NAME, NR} given by \Rind{GSPOS}
\item[ISEL=7] to modify {\tt NDIV} given by \Rind{GSDVN}
\item[ISEL=8] to modify {\tt IAXIS} given by \Rind{GSDVN}
\end{DL}
